# Blog Images

Place all blog-related images here.

## Image Specs

**Cover Images (placeholder-blog.jpg and post covers)**
- Resolution: **1600x900px** (16:9 ratio)
- Format: JPG for photos, PNG for graphics
- Quality: 75-85% compression
- Max size: 300KB

**Content Images (used inside blog posts)**
- Width: **1200px** (height varies)
- Format: JPG/PNG depending on content
- Max size: 200KB

## Current Files

- `placeholder-blog.png` - Default cover image for posts without custom covers

## Naming Convention

Use descriptive, lowercase names with hyphens:
- `digital-minimalism-cover.jpg`
- `creative-systems-diagram.png`
- `visual-thinking-hero.jpg`
